# Travel Weekend Itinerary App ✈️

## 🚀 Live Demo
https://siyadsheikh.github.io/Web-app/

## 📌 About the Project
This is a simple web app that is about a weekend travel itinerary.
The goal of this project was to practise JavaScript fundamentals and build a useful, real-world style application.

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## ✨ Features
- Add travel activities or plans
- Organise itinerary by day (e.g. Friday, Saturday, Sunday)
- Delete or update activities
- Simple and clean user interface


## 🧠 What I Learned
- How to handle user input and events
- How to structure a multi-section web app
- Improving layout and responsiveness with CSS
